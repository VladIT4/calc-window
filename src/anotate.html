Тут редактируются группы окон. Это техническая часть, на цены она не влияет. Группы, это окна из левого меню. Каждая група, это отдельный главный пункт.

Имеются следующие поля

  • max_width: Максимальная ширина окна
  • min_width: Минимальная ширина окна
  • max_height: Максимальная высота окна
  • max_height: Минимальная высота окна
  • name: Отображаемое имя для окна

Меню содержит под-пункты, в поле items

        {
            "type": Тип окна,
            "count_choice": кол-во створок в окне,
            "slug": техническое поле для идентификации,
            "default": {
                значения створок поумолчанию. 
                Содержит значения для параметров 
                "left", "center", "right" - левого, 
                центрального и правого окна
            }
        }
[ { "groups": [ { "name": "Одностворчатое окно", "slug": "window1", "items": [ { "type": "Одностворчатое", "count_choice": 1, "slug": "window1_0", "default": { "center": "глухое" } }, { "type": "Одностворчатое", "count_choice": 1, "slug": "window1_0", "default": { "center": "поворотное-Л" } }, { "type": "Одностворчатое", "count_choice": 1, "slug": "window1_0", "default": { "center": "поворотное-откидное-Л" } } ] }, ............

Цены за створки

Каждый блок цены отвечает за параметры цены для определенной створки, имя которого содержится в параметре name

В параметре prices содержатся цены на выбранную створку с выбранным профилем

                    {
                        "name": "Название",
                        "prices": {
                            "профиль": цена
                        }
                    },
"prices": [ { "name": "глухое", "prices": { "KBE": 1000, "Rehau": 1500, "Trocal": 2000 } }, { "name": "поворотное-Л", "prices": { "KBE": 1000, "Rehau": 1500, "Trocal": 2000 }, "mosquito": 10 }, ............ ],

Другие параметры калькулятора

"other": {

Фиксированная цена за "комплектующие". Если цену не указать, галочки "комплектующие" не будет.

"Комплектующие": 2222,

Цена за кв. метр.

"Стоимость монтажа": 1500,

Цена за погонный метр

"Стоимость отделки под ключ": 300,

Скидка в процентах на итоговую стоимость в зависимости от выбранного профиля

"Скидка": { "KBE": 50, "Rehau": 19, "Trocal": 0 },

Цена за ламинацию. Цена устанавливается для всех цветов сразу.

"ЛаминацияЦена": 1000,

Список цветов для ламинации. Первый параметр название цвета, второй параметр цвет в HEX RGB
